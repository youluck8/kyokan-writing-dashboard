# 共感ライティング 入会状況ダッシュボード

継続者・新規の入会状況とマーケ施策スケジュールをリアルタイムで共有するための静的サイト。

## 仕組み

- データはこのGoogleスプレッドシートに書き込む: https://docs.google.com/spreadsheets/d/15DN8OKsj7vlnj1AL9RbIF2M-SchyMYdA_iSloX6GZq0/edit
- サイトはスプレッドシートを自動で読み込んで表示する（1分ごとに自動更新）
- 行を自由に追加してOK。列: `更新日, カテゴリ(継続/新規/共通), 項目, 現状, 今後の施策, スケジュール, 担当, メモ`

## 重要: スプレッドシートの共有設定

サイトから読み込めるようにするため、スプレッドシートを「リンクを知っている全員が閲覧可」に設定してください（共有ボタン → 一般アクセス → リンクを知っている全員 → 閲覧者）。書き込む人には別途編集者として招待してください。

## サイトのパスワード

トップページは一見「404 Not Found」に見えますが、どこでもよいのでパスワードを入力すると中身が表示されます。パスワードは `script.js` 内の `PASSWORD` で変更できます。
